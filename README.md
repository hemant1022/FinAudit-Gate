# FinAudit-Gate

Adaptive Multi-Agent Financial Portfolio & Risk Auditor with GRPO training.

## Architecture

1.  **Data Ingestion**: SEC EDGAR, Yahoo Finance, News APIs.
2.  **Analysis Engine**: Financial health, valuation, portfolio optimization.
3.  **Risk Auditor**: Guardrail validation, stress testing, factual verification.
4.  **Cyclic Feedback Loop**: LangGraph State Machine with Analyst -> Auditor loops and GRPO Training.

## Phase 0 Setup

This repository contains the setup for Phase 0 of the project.
